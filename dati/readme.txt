Qui potete trovare i file che mi sono stati forniti:
- i file .shp e .csv contengono rispettivamente la geometria (coordinate GPS e riferimenti) e le informazioni sulle piante
- il file geojson è invece un'elaborazione tramite QGIS dove ho provato a mettere insieme i due file

Noterete che ci sono diversi problemi:
1. i due file non corrispondono per numero di voci (utilizzando l'id pianta possiamo vedere che nel file .shp troviamo alcuni codici piante ripetuti)
2. nella colonna diametro pianta sono stati inseriti dei valori che incasinano la tipologia di dato (invece che inserire valori numerici ci sono anche valori testuali)
3. ci sono alcune piante già segnalate per l'abbattimento, c'è da capire come segnalarle/se segnalarle (es. visibili solo nella mappa manutentiva)
